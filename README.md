# MediCore - Hospital Management System

## Description
MediCore is a Hospital Management System built using Django and Django REST Framework. 
It provides secure REST APIs for managing doctors, patients, and appointments.

## Features
- JWT Authentication
- Secure REST APIs
- MySQL Database Integration
- Admin Panel Management
- CRUD Operations

## Technologies Used
- Python
- Django
- Django REST Framework
- MySQL
- JWT Authentication

## API Endpoints
- /api/doctors/
- /api/patients/
- /api/appointments/
- /api/token/
- /api/token/refresh/
